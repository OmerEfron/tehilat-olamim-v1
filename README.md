# תהילת עולמים · Tehilat Olamim

A single-player card guessing game. Pure frontend — no backend.

## Play locally

```bash
npm install
npm run dev
```

## Static build

```bash
npm run build
```

Output lands in `out/`. Host that folder on any static host (GitHub Pages, Cloudflare Pages, S3, Netlify, etc.).
